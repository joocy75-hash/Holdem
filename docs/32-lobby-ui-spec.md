```

```

│  [🔍 방 이름 검색...]                                        │

## 3. 방 생성 모달

### 3.1 레이아웃

```
┌─────────────────────────────────────────────────────────────┐
│  방 만들기                                            [✕]   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  방 이름                                                    │
│  [초보자 테이블                                    ]        │
│                                                             │
│  블라인드                                                   │
│  [10/20 ▼]                                                 │
│                                                             │
│  최대 인원                                                  │
│  [2] [6 ●] [9]                                             │
│                                                             │
│  바이인 범위                                                │
│  Min: [400    ]  Max: [2000   ]                            │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                              [취소]  [방 만들기]            │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 입력 검증

| 필드 | 규칙 | 에러 메시지 |
|------|------|-----------|
| 방 이름 | 2-20자 | "방 이름은 2-20자여야 합니다" |
| 바이인 Min | ≥ 20 BB | "최소 바이인은 20 BB 이상이어야 합니다" |
| 바이인 Max | ≤ 200 BB | "최대 바이인은 200 BB 이하여야 합니다" |
| Min ≤ Max | Min ≤ Max | "최소 바이인이 최대보다 클 수 없습니다" |

## 4. 방 입장 플로우

### 4.1 입장 확인 모달

```
┌─────────────────────────────────────────────────────────────┐
│  방 입장                                              [✕]   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  초보자 테이블 #1                                           │
│  Blinds: 10/20 | Players: 4/6                              │
│                                                             │
│  바이인 금액                                                │
│  [────────●────────]                                       │
│  $1,000                                                    │
│  (Min: $400 ~ Max: $2,000)                                 │
│                                                             │
│  현재 잔액: $5,000                                          │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                              [취소]  [입장하기]             │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 잔액 부족 시

```
┌─────────────────────────────────────────────────────────────┐
│  ⚠️ 잔액 부족                                               │
│                                                             │
│  최소 바이인 $400이 필요합니다.                              │
│  현재 잔액: $200                                            │
│                                                             │
│                                    [확인]                   │
└─────────────────────────────────────────────────────────────┘
```
