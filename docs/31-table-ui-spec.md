## 1. 테이블 레이아웃

### 1.1 전체 구조

```
┌─────────────────────────────────────────────────────────────┐
│  [← 로비] [초보자 테이블 #1] [10/20]           [⚙️] [💬]    │  Header
├─────────────────────────────────────────────────────────────┤
│                                                             │
│              [Seat 3]              [Seat 4]                 │
│                 👤                    👤                    │
│              $1,200                 $800                    │
│                                                             │
│     [Seat 2]                              [Seat 5]          │
│        👤          ┌──────────────┐          👤             │
│      $1,500        │  [♠A][♥K][♦Q]│        $950            │
│                    │              │                         │
│                    │  Pot: $200   │                         │
│     [Seat 1]       │    [D]       │      [Seat 6]          │
│        👤          └──────────────┘          👤             │
│      $2,000                               $1,100            │
│                                                             │
│                      [Seat 0 - ME]                          │
│                         👤 ★                                │
│                       $1,500                                │
│                      [♠A][♠K]                               │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  [Fold] [Check] [Call $20] [Raise ▼]           ⏱️ 0:25     │  Action
└─────────────────────────────────────────────────────────────┘
```

## 2. 좌석 컴포넌트

### 2.1 좌석 상태별 표시

#### 빈 좌석
```
┌─────────────┐
│     ➕      │
│   착석하기   │
└─────────────┘
```

#### 플레이어 착석
```
┌─────────────┐
│     👤      │
│  Player1    │
│   $1,500    │
│  [Bet: $20] │
└─────────────┘
```

#### 현재 턴
```
┌─────────────┐  ← 하이라이트 테두리
│     👤      │     + 타이머 표시
│  Player1 ⏱️ │
│   $1,500    │
│  [Bet: $20] │
└─────────────┘
```

#### 폴드
```
┌─────────────┐  ← 흐리게 처리
│     👤      │
│  Player1    │
│   $1,500    │
│   [FOLD]    │
└─────────────┘
```

#### 올인
```
┌─────────────┐  ← 강조 표시
│     👤      │
│  Player1    │
│   ALL-IN    │
│   [$1,500]  │
└─────────────┘
```

### 2.2 좌석 정보

| 요소 | 표시 조건 | 위치 |
|------|----------|------|
| 아바타 | 항상 | 상단 |
| 닉네임 | 항상 | 아바타 아래 |
| 스택 | 착석 시 | 닉네임 아래 |
| 베팅액 | 베팅 시 | 하단 |
| 딜러 버튼 | 딜러 위치 | 좌석 옆 |
| 타이머 | 현재 턴 | 닉네임 옆 |
| 홀카드 | 쇼다운 시 | 좌석 아래 |

## 3. 커뮤니티 카드 영역

### 3.1 레이아웃

```
┌─────────────────────────────────────────┐
│                                         │
│    [♠A] [♥K] [♦Q] [  ] [  ]            │  ← 카드 5장
│                                         │
│              Pot: $200                  │  ← 팟 금액
│                                         │
│         [D] ← Dealer Button             │
│                                         │
└─────────────────────────────────────────┘
```

### 3.2 카드 표시

| Phase | 표시 카드 |
|-------|----------|
| Preflop | 없음 |
| Flop | 3장 |
| Turn | 4장 |
| River | 5장 |
| Showdown | 5장 + 승자 하이라이트 |

### 3.3 카드 애니메이션

| 이벤트 | 애니메이션 |
|--------|-----------|
| 카드 딜 | 슬라이드 인 |
| 카드 오픈 | 플립 |
| 승리 카드 | 하이라이트 펄스 |

## 4. 액션 패널

### 4.1 기본 레이아웃

```
┌─────────────────────────────────────────────────────────────┐
│  [Fold]  [Check]  [Call $20]  [Raise ▼]        ⏱️ 0:25     │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 액션별 버튼 상태

| 액션 | 활성화 조건 | 버튼 텍스트 |
|------|-----------|-----------|
| Fold | 항상 | "Fold" |
| Check | 콜 금액 = 0 | "Check" |
| Call | 콜 금액 > 0 | "Call $XX" |
| Bet | 첫 베팅 | "Bet" |
| Raise | 이전 베팅 있음 | "Raise" |
| All-in | 스택 ≤ 콜 금액 | "All-in $XX" |

### 4.3 레이즈 슬라이더

```
┌─────────────────────────────────────────────────────────────┐
│  Raise Amount                                               │
│  [Min]  [────────●────────]  [Max]                         │
│         $100                                                │
│                                                             │
│  [1/2 Pot] [Pot] [2x Pot] [All-in]                         │
│                                                             │
│                              [Cancel]  [Raise $100]         │
└─────────────────────────────────────────────────────────────┘
```

### 4.4 버튼 비활성화 조건

| 조건 | 비활성화 버튼 |
|------|-------------|
| 내 턴 아님 | 모든 버튼 |
| 재연결 중 | 모든 버튼 |
| 액션 처리 중 | 모든 버튼 |
| 스택 부족 | Raise |

## 5. 타이머

### 5.1 표시 위치

- 현재 턴 플레이어 좌석 옆
- 액션 패널 우측

### 5.2 타이머 상태

| 남은 시간 | 색상 | 효과 |
|----------|------|------|
| > 10초 | 녹색 | 없음 |
| 5-10초 | 노란색 | 없음 |
| < 5초 | 빨간색 | 깜빡임 |
| 0초 | - | 자동 폴드 |

### 5.3 타이머 UI

```
일반: ⏱️ 0:25
경고: ⏱️ 0:08 (노란색)
긴급: ⏱️ 0:03 (빨간색, 깜빡임)
```

## 6. 쇼다운 UI

### 6.1 결과 표시

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                      🏆 WINNER 🏆                           │
│                                                             │
│     Player1                              Player3            │
│     [♠A][♠K]                            [♥Q][♥J]           │
│     Flush, Ace High                     Two Pair            │
│     +$200                               -                   │
│                                                             │
│                    [다음 핸드 시작: 5초]                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 6.2 핸드 랭크 표시

| 랭크 | 한글 | 영문 |
|------|------|------|
| Royal Flush | 로열 플러시 | Royal Flush |
| Straight Flush | 스트레이트 플러시 | Straight Flush |
| Four of a Kind | 포카드 | Four of a Kind |
| Full House | 풀하우스 | Full House |
| Flush | 플러시 | Flush |
| Straight | 스트레이트 | Straight |
| Three of a Kind | 트리플 | Three of a Kind |
| Two Pair | 투페어 | Two Pair |
| One Pair | 원페어 | One Pair |
| High Card | 하이카드 | High Card |
